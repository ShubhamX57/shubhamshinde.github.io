★ Capstone Neutron Physics Bayesian MCMC FastAPI Open Source
★QENS Analysis & Visualisation Suite
MSc Capstone · University of Bristol · Aligned with ISIS Neutron Source workflows
2025 — present

Open-source pipeline for quasi-elastic neutron scattering (QENS) analysis — my MSc capstone at Bristol. Implements the full physics chain from raw HDF5 experimental files through resolution correction, model fitting, and Bayesian parameter estimation, presenting results through a FastAPI + React web interface.

The core physical model is the Chudley-Elliott jump-diffusion framework. Getting the resolution-function deconvolution right was the decisive technical challenge — a naïve approach inflates the diffusion coefficient by ~20%.

eq. 1 — Chudley-Elliott structure factor
$$S(Q,\omega) = \frac{1}{\pi} \cdot \frac{\Gamma(Q)}{\omega^2 + \Gamma^2(Q)}$$
where $\Gamma(Q) = D Q^2 / (1 + Q^2 l^2)$, $D$ = diffusion coefficient, $l$ = jump length
eq. 2 — Resolution-corrected spectrum
$$I(Q,\omega) = S(Q,\omega) \otimes R(Q,\omega) + B(Q)$$
repository
github.com/ShubhamX57/QENS
test results — passing
  • 16% reduction in systematic error in diffusion coefficient via physics-correct resolution correction
  • 3× improvement in χ² goodness-of-fit: Chudley-Elliott vs Fickian model
  • Bayesian MCMC across 4 chains, Gelman-Rubin $\hat{R} < 1.01$ verified convergence
  • FastAPI + React web interface in active development
Fig. 1  QENS spectrum: data + Chudley-Elliott fit · χ²/dof ≈ 1.04
The 3× χ² improvement over Fickian is the result I'm proudest of — it means the physics model actually matters, not just the software.
